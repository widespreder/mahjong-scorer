# 🀄 麻雀スコアラー

ブラウザだけで動く、かんたん麻雀点数計算 ＆ 対局管理アプリ。

**👉 [ライブデモ](https://YOUR_GITHUB_USERNAME.github.io/mahjong-scorer/)**（公開後にURLを更新）

## 主な機能

- **1局計算**: 翻・符から点数を計算（役選択で自動算出）
- **対局管理**: 半荘戦／東風戦の進行・連荘・本場・リーチ棒・ノーテン罰符を全自動処理
- **役一覧**: 42役を分類表示、読み仮名つき
- **翻数クイズ**: 役の翻数を覚えるための練習モード
- **対局履歴**: localStorage に自動保存（ブラウザを閉じても残る）

## 対応ルール

- 半荘戦 / 東風戦
- 連荘（テンパイ連荘 / あがり連荘）
- 食いタン・後付け
- 飛び（トビ）終了
- アガリやめ
- 連風牌雀頭 +4符
- 持ち点・返し点の調整

## 特殊役の符自動セット

- 平和ツモ → 20符固定
- 七対子 → 25符固定
- 鳴きロン → 基本20符
- ダブル役満（小四喜+字一色など）→ 単役満の2倍の点数

## 技術スタック

- React 18 (CDN, スタンドアロン)
- Babel スタンドアロン
- 単一HTMLファイル / ビルド不要
- データ保存: localStorage

## ローカルで動かす

ファイルをブラウザで開くだけで動きます。

```bash
# Pythonがあれば簡易サーバを立てて閲覧
python -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```
